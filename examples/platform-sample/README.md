# platform-sample

Small example showing how to compose the atomic modules directly.

## Purpose

This example is intentionally simple and demonstrates:

- resource group
- virtual network
- subnets
- private DNS zone
- private DNS zone link
- SQL server
- SQL database
- SQL private endpoint

## Notes

- This example uses the atomic modules under `modules/`.
- It is not intended to represent the full platform.
- A separate example can be added later for `stacks/`.

## Usage

Run it from inside this folder with a valid AzureRM provider configuration in place.
